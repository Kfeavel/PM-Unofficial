    ルイージ

    キャラクター紹介

    永遠の２番手は、DXでおなじみの動きといくつかの新しいテクニックを引っさげて帰ってきました。 長い間、兄であるマリオの影に隠れていたことで、ルイージはマリオとの差別化が進んでいます。 類似しているモーションであるにもかかわらず性能が全く違うことから、兄弟であってもその性能の差は歴然としていました。 Project Mでは、DXの動きを取り戻し、さらにわずかな調整を加えました。 このキャラクターで対戦すれば、DX随一の滑り性能を体感できるでしょう！もう２番手とは言わせません！


    滑りの達人

    絶と組み合わせることで異常な使い勝手を発揮する横強攻撃は、DXのモーションと性能を取り戻しました。 圧倒的なスピードでのコンボの追撃や、電光石火の速さで相手の懐に潜り込む性能を持っています。 さらに、遥かに強力な立ち回りを行えるように小さな改善点があります。横強攻撃のほかに空中前攻撃、空中上攻撃、ファイアボールなどのメインとなる技のいくつかは攻撃範囲が大きくなり、いくつかの状況でより多くダメージを与えられます。 パワフルな下強攻撃に関して、話すことはないでしょう...


    ミサイルの制御

    今までのルイージロケットはランダムに強力な暴発が発生していましたが、今はこの暴発を制御することが出来ます。 Project Mではもう遊び技ではありません。 ルイージロケットを使うと、バレルを回すロシアンルーレットのように"６発"以内に暴発が発生します。 暴発前には、ルイージが緑色に点滅します。 ルイージが発射される前にＬＲボタンを入力すると、ルイージロケットは通常の攻撃に戻り、次の攻撃に暴発を先送りすることができます。何度でも暴発を先送りすることが出来るので暴発を自由に制御することができ、不慮の暴発での致命的なミスを防ぐことが出来ます。


    高い復帰力

    DXでのルイージの弱点の一つが、重力に見合わない難しい復帰でした。 弱い判定と低い空中機動力も相まって、低ベクトル攻撃に対しては無力と言わざると得なかったのです... ステージに戻るためには多大な集中力が必要で、崖を守る相手には復帰の難易度はさらに跳ね上がります。 Project Mは、この弱点を復帰技の後隙を減らすことによって解決しています。 ルイージロケットは崖を直接掴むことができ、ファイアボールは復帰の手助けに十分な性能で、スーパージャンプパンチ後の空中横移動力はそこまで落ちません。 さらに重要なのは、ルイージサイクロンは今まで通り復帰に使用することができ、ステージ上での"チャージ"を必要としなくなりました。 また、以前より少ないボタン連打で上昇することができ、技が終わったあとも以前より早く行動できます。 ルイージは以前より復帰しやすくなったのです。
